		<h1 class="entry-title">Data Analytics</h1>	</header><!-- .entry-header -->

	<div class="entry-content">
		
<p class="has-text-align-center"><img width="700" height="500" src="https://digitalcreativemind.com/wp-content/uploads/2021/06/Analytics_amp_Data_Science.gif"></p>



<div class="wp-block-image"><figure class="alignright size-large"><img data-attachment-id="2213" data-permalink="https://limitlessdatascience.wordpress.com/data-science-process-1/" data-orig-file="https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png" data-orig-size="372,213" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="data-science-process-1" data-image-description="" data-image-caption="" data-medium-file="https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png?w=300" data-large-file="https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png?w=372" width="372" height="213" src="https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png?w=372" alt="" class="wp-image-2213" srcset="https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png 372w, https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png?w=150 150w, https://limitlessdatascience.files.wordpress.com/2019/11/data-science-process-1.png?w=300 300w" sizes="(max-width: 372px) 100vw, 372px" /><figcaption> Dig: Major steps in Data Science</figcaption></figure></div>



<p class="has-text-align-justify">In this article, I am going to explain some of the important steps generally we follow in most Data Science Project development. Data analysis is done to solve problems or problems from the <strong>insights </strong>gained after processing available data. </p>



<p>In this article we will take a look at some of the steps generally we take during data analysis. These steps are explained below.</p>



<div class="wp-block-image"><figure class="aligncenter size-large"><img data-attachment-id="1956" data-permalink="https://limitlessdatascience.wordpress.com/analysis-1/" data-orig-file="https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png" data-orig-size="1008,75" data-comments-opened="0" data-image-meta="{&quot;aperture&quot;:&quot;0&quot;,&quot;credit&quot;:&quot;&quot;,&quot;camera&quot;:&quot;&quot;,&quot;caption&quot;:&quot;&quot;,&quot;created_timestamp&quot;:&quot;0&quot;,&quot;copyright&quot;:&quot;&quot;,&quot;focal_length&quot;:&quot;0&quot;,&quot;iso&quot;:&quot;0&quot;,&quot;shutter_speed&quot;:&quot;0&quot;,&quot;title&quot;:&quot;&quot;,&quot;orientation&quot;:&quot;0&quot;}" data-image-title="analysis-1" data-image-description="" data-image-caption="" data-medium-file="https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=300" data-large-file="https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=740" width="1008" height="75" src="https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=1008" alt="" class="wp-image-1956" srcset="https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png 1008w, https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=150 150w, https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=300 300w, https://limitlessdatascience.files.wordpress.com/2019/10/analysis-1.png?w=768 768w" sizes="(max-width: 1008px) 100vw, 1008px" /><figcaption>Dig: Analyse the prepared data</figcaption></figure></div>



<h3 class="has-vintage-blue-color has-text-color wp-block-heading">STEP 1: Understand the data:</h3>



<p class="has-text-align-justify">Many of the time new developers ignore or make assumptions regarding data (about quality, definitions of the fields or columns or features) and with that, they start to develop a code and when they got stuck somewhere they spent a lot of time finding problems in their code and later find out the problem was in the data itself :D. During this process, they waste their valuable time and resources. </p>



<p class="has-text-align-justify">Hence Studying data is the most important thing to do in the Data Science. Real-world data is really messy. Since lots of impurities may get added during the journey of data from collection to storage and back to analysis. You have to make sure your data is good to go for the analysis.   </p>



<p class="has-text-align-justify">First you should know how data was collected (briefly the process)? How data was stored (platform, format used)? What was the purpose of data collection(for the Organisation)? Definition of each field in the data?  Verify the assumption made about any data fields in the data. Getting precise answers and recording them for future reference as the dictionary is the most important thing that as Data Scientists/Analysts we should do. Here we need the help of <span style="text-decoration:underline;">domain experts or domain knowledge</span> to verify our assumptions, definitions, etc about data.</p>



<p class="has-text-align-justify">This kind of data investigation will help in every stage of project development from Pre-processing to finally gain insights from the output. After getting this information one can start to study the data to find any gaps in it and to get more general information about the data and the fields in it.</p>



<p class="has-text-align-justify">Let&#8217;s understand this with simple example. For the explanation purpose I have implemented the following techniques to study the data: </p>



<ul><li>By getting Basic information about data using pandas functions</li><li>By using using graphs using Pandas plot() function, </li><li>By finding any gaps, outliers using box plot, dbscan clustering method</li></ul>



<p class="has-text-align-justify">To see the example implementation refer to this  <a rel="noreferrer noopener" href="https://github.com/Yogeshnaik1190/Analytics/blob/master/1_Study_Input_Data.ipynb" target="_blank"><strong>GitHub repository</strong></a> having example code, data.  </p>



<p class="has-text-align-justify">A similar list of techniques you can use to check data quantity, quality, or gaps of new input data. Note that this list of techniques will increase or decrease as per project requirements and also with your experience with time.</p>



<h3 class="has-vintage-blue-color has-text-color wp-block-heading">STEP 2 Handle missing or empty or Nan data</h3>



<p class="has-text-align-justify">Remove such data points or fields or features if that is not going to impact the algorithm or not going to contribute to solving the problem.</p>



<p class="has-text-align-justify">But what if that is important and will be required to make judgments or will be used as the input feature in our algorithm. Then in such a case removal will not work. In such a case we can use some of the imputing techniques. I have explained a few of the methods in the following article:</p>



<ul><li><a href="https://limitlessdatascience.wordpress.com/2019/09/18/time-series-imputing/" target="_blank" rel="noreferrer noopener" aria-label="Data imputing using interpolate, fillna - Time series example (opens in a new tab)"><strong>Data imputing using interpolate, fillna &#8211; Time series example</strong></a></li></ul>



<p class="has-text-align-justify">Imputing should be done in such a way so that it will contribute to improving the output.</p>



<h3 class="has-vintage-blue-color has-text-color wp-block-heading">STEP 3: In depth Data analysis, Derive new input features</h3>



<p class="has-text-align-justify">Advanced analytics allows us to deep dive into data. This can be used to derive new features from the existing raw data columns/fields. </p>



<p class="has-text-align-justify">Now we will take a look at some of the useful techniques: </p>



<ol><li><strong><a rel="noreferrer noopener" href="https://limitlessdatascience.wordpress.com/2019/02/08/pandas-plot-function-exploration/" target="_blank">Exploring plot() function&#8217;s useful features in depth</a></strong>  </li><li><strong>Detailed analysis</strong> <strong>with Pandas functions for analysis: Pivot table</strong>,  <strong>Group-by and merge, melt</strong>  (coming soon)</li></ol>



<p class="has-text-align-justify">These derived features can be used to gain further insights. Note that <span style="text-decoration:underline;">many of the times its most likely that you may find or end up deriving features and applying simple statistical techniques to them can solve the problems that we are required to solve instead of moving to deep learning or machine learning algorithms to solve them</span>. </p>



<ul><li>Therefore always be open or flexible enough to think for the possible solutions other than machine learning and deep learning or mix of both before starting the project or during your project development. </li></ul>



<p><strong>Data Preparation:</strong></p>



<p class="has-text-align-justify">Now suppose that we have prepared the input table for our algorithm but we want to further study the prepared features then we can use the following steps for that purpose. </p>



<ul><li><strong>Correlation Matrix</strong> <ul><li>Find out correlation between different input features. Does it make sense to use them as the input to the machine learning algorithm? </li><li>For more explanation refer this <a rel="noreferrer noopener" aria-label="article (opens in a new tab)" href="https://limitlessdatascience.wordpress.com/2019/01/01/correlation-matrix-for-different-input-features/" target="_blank"><strong>article</strong></a></li></ul></li><li><strong>Find out causality</strong>. <ul><li>If in the output we found a particular pattern or behaviour then try to find the factors causing such a behaviour or pattern in the output. E.g. Users visiting a store are in the Age group of 15-50. But we found out Users with Age 20 end up as a buyers more frequently than Users with Age 30. Why? There can be some causing factor. </li></ul></li></ul>



<h3 class="has-vintage-blue-color has-text-color wp-block-heading">STEP 4: Study Output. Analysis to gain insights from the data</h3>



<p class="has-text-align-justify">We know the main goal of data analysis is to gain insights from the data. And it&#8217;s not necessary that all the insights will be gained at the end of all the above steps from 1 to 3. Most of the time some of the important insights from the data can be learned during the implementation of the above three steps.</p>



<p class="has-text-align-justify">Hence after every step of implementation from pre-processing to testing in data science always question what we can see from the results. Our job is not just implementing techniques and tools to generate outputs from the data but it&#8217;s equally important to decode or understand that output to gain important insights. And that can be done with careful investigation and study of the output.</p>



<h3 class="has-vintage-blue-color has-text-color wp-block-heading">Is it true that insights gained from the data always solves the problems that data scientists supposed to solve?</h3>



<ul><li>Simple and straight answer is <strong>NO</strong>. </li><li>It&#8217;s very true there are many problems that industries or organizations currently facing which initially thought can completely be solved using data science alone but in current reality if we see the most of the data science projects are not able to solve the 100% problems.  </li><li>It&#8217;s most likely that you may end up doing some traditional hard coded rule or heuristics based solutions along with Data science techniques. This is because of the limitation of data, data analysis techniques, available resources (human resources, machines-computation power, finance, domain expertise) and time or any other reasons.</li><li>Therefore it&#8217;s most likely that In Data Science projects you may end up doing hybrid solutions which have some or most problems tackled using data science techniques and some using rule based or heuristics based solutions.</li><li>Note that It&#8217;s possible that data science projects may end up in failure that not able to solve problems faced at all.  Therefore from the start and during development steps of the project always ask whether project is leading to the solutions that we want or should we stop it to avoid wastage of resources. </li></ul>



<p class="has-text-align-justify">This is it. I tried to keep this article as small as possible. I hope this will help you to learn some useful steps in data analytics. </p>
